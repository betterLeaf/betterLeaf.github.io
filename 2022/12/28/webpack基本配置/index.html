webpack基本配置

Created2022-12-28|Updated2022-12-28
|Post View:

mode

根据环境 判断

entry 入口文件

output

输出文件

  • filename 打包的同步文件
  • chunkFilename 打包的异步文件
  • publicPath 静态资源添加路径

loader

loader都是写到 module.rules数组中

处理css

loader (style-loader, css-loader)

处理scss

  • sass-loader：负责将 Sass 文件编译成 css 文件
  • sass：sass-loader 依赖 sass 进行编译

分离css代码

miniCssExtractPlugin 此插件 和style-loader插件有冲突，使用替换掉style-loader

plugins

htmlWebpackPlugin

  • 改变页面title
  • 使用自定义的html
  • publicPath

clean-webpack-plugin

每次打包，会删除旧的打包文件

其他配置项

webpack.config.js

webpack默认的配置文件为 webpack.config.js，但是我们可以修改此文件名 如 webpack.dev.js,此时我们需要在脚本命令中 –config 配置一下
