this

Created2022-12-26|Updated2022-12-27
|Post View:

结论

  • new优先级最高
  • bind次之
  • obj.func()
  • 最后是func()
  • 箭头函数的this一旦绑定，不会被任何方式改变
  • 立即执行函数和非同步函数setTimeout大多指向全局
  • return 和this (返回值是一个对象,this指向那个返回对象，如果返回值不是一个对象，this还是指向函数的实例，null虽然也是对象但还是指向函数实例)

直接调用函数

直接调用函数，不管函数在什么地方, this一定指向win

对象调用

对于obj.fun()来说，this永远指向最后调用它的对象

new

对于new来说，this永远绑定在实例上，不会被任何方式改变

return 和this

(返回值是一个对象,this指向那个返回对象，如果返回值不是一个对象，this还是指向函数的实例，null虽然也是对象但还是指向函数实例)

箭头函数

箭头函数没有this，取决于包裹他的第一个普通函数的this，箭头函数无法用bind

bind

bind取决于第一个参数，如果为空，那么就是win, 多次bind，永远是第一次bind决定
